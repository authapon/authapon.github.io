การลงระบบ X Windows และ Cinnamon Desktop ใน Archlinux

Sunday, 24 August 2014



ในระบบ Linux นั้นจะมี Desktop ให้เลือกใช้มากมาย แต่วันนี้ผมจะมาแนะนำ Desktop ที่ผมชอบ ซึ่งก็คือ Cinnamon เนื่องจากเป็น Desktop ที่เรียบง่าย ปรับแต่งความสวยงามได้ตามความชอบ โดยเริ่มง่าย ๆ ดังนี้ครับ

1. ลง package ที่เกี่ยวข้อง

เราต้องลง package ที่เกี่ยวข้องกับ X Windows, Cinnamon Desktop, fonts และ โปรแกรมที่ใช้ ดังนี้ครับ

2. แก้ไขไฟล์ /etc/X11/xinit/xinitrc

จากนั้นเราก็ทำการแก้ไขไฟล์ /etc/X11/xinit/xinitrc ดังนี้ครับ

File : /etc/X11/xinit/xinitrc

#twm &
#xclock -geometry 50x50-1+1 &
#xterm -geometry 80x50+494+51 &
#xterm -geometry 80x20+494-0 &
#exec xterm -geometry 80x66+0+0 -name login

exec cinnamon-session

3. ทำการเรียก Cinnamon Desktop

เมื่อเราทำการลง package และปรับแต่งไฟล์ xinitrc แล้ว เราก็สามารถเข้าสู่ โหมด Desktop ได้ด้วยคำสั่ง startx ดังนี้ครับ

เพียงเท่านี้ เราก็เข้าสู่โหมด Desktop แล้วครับ :)
