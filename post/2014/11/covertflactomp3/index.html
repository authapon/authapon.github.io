การแปลงไฟล์เสียง Flac ไปเป็น MP3 แบบคุณภาพสูง

Wednesday, 19 November 2014



ผมได้ทดลองแปลงไฟล์เสียงคุณภาพสูง Flac ไปเป็น MP3 แบบคุณภาพสูงในระบบ Archlinux แบบง่าย ๆ โดยสามารถทำได้ดังนี้ครับ

1. ลง package ที่จำเป็น

เราต้องลง package ที่จำเป็นในการแปลงไฟล์เสียงคือ ffmpeg และลง package ที่ช่วยที่ทำงานแบบขนานได้สำหรับเครื่องหลาย CPU คือ parallel ดังนี้ครับ

2. สร้าง script สำหรับการแปลงไฟล์

เราสร้างไฟล์ mp3.sh สำหรับการแปลงไฟล์ดังนี้

File : mp3.sh

#!/bin/bash

parallel -j4 ffmpeg -i {} -qscale:a 0 {.}.mp3 ::: *.flac

จากนั้นในการใช้งานก็เพียงเรียก script นี้ตอนอยู่ในโฟล์เดอร์ที่ต้องการ script ก็จะทำการแปลงไฟล์ Flac ทั้งหมดให้เป็น MP3 ครับ
